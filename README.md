## 关于项目 ##

    作者： Ken Ma
    启动日期： 2017-5-1
    说明：本项目是Angular上的分页directive，特点是代码简洁易懂，使用者可以根据自己的需要增加效果。
    
    
## 使用 ##
    1.  引入bootstrap.css样式文件。也可以自己编写样式。
    2.  引入pagination.js脚本文件。此指令没有对外开放的函数，只有两个需要双向绑定的变量page和maxPage。
    3.  通过读取page可以获得当前页码。
    4.  指令内部对maxPage进行了监听，一旦此变量变化，就会执行html页码数量的初始化。
